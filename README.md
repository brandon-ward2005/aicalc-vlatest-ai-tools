## Overview

aicalc is a browser-oriented set of AI developer utilities that keeps routine tasks in one place. The app is built around client-side execution, so you can handle quick calculations and generators without sending every action through a remote service.

It is built for developers dealing with LLMs, security headers, privacy-conscious text processing, and general environment cleanup. Inside one interface, it combines tools for pricing estimates, token counting, cron expression generation, PII redaction, dependency review, secret scanning, and more.

1. Launch the app in your browser.
2. Select the utility you need, such as pricing, token counting, or cron generation.
3. Enter or paste your input.

- Check a dependency list for known issues
